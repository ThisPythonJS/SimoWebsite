## Sobre o Projeto

O Simo Botlist é um website desenvolvido para que desenvolvedores possam expor seus bots do Discord, facilitando a descoberta por novos usuários. O projeto foca em uma interface limpa, performance rápida e facilidade de navegação.

### Principais Características
- Listagem de bots: Exibição organizada de diferentes aplicações.
- Interface Responsiva: Adaptada para dispositivos móveis e desktop.
- Integração Frontend: Desenvolvido com foco em componentes reutilizáveis e estados tipados.

---

## Stack Tecnológica

As principais tecnologias utilizadas no desenvolvimento deste projeto foram:

- React: Biblioteca principal para construção da interface.
- TypeScript: Utilizado para garantir maior segurança e previsibilidade do código.
- Vite: Ferramenta de build de última geração para um desenvolvimento rápido.
- Tailwind CSS: Framework de estilização via classes utilitárias.
- Animate.css / Framer Motion: Implementação de transições e animações visuais.

## Como Rodar o Projeto Localmente

Siga as instruções abaixo para configurar o ambiente em sua máquina:

### Pré-requisitos
- Node.js instalado
- Gerenciador de pacotes (NPM ou Yarn)

### Passo a Passo

1. Clone este repositório:
```bash
git clone https://github.com/ThisPythonJS/SimoWebsite.git
cd SimoWebsite
```

2. Instale todas as dependências necessárias:
```bash
bun install
```

3. Inicie o servidor de desenvolvimento:
```bash
bun dev
```

4. Acesse o projeto:
O site estará disponível em `http://localhost:5173`.
